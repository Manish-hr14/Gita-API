## Getting Started

First, setup all the ENVs on .env.local file as on .env.example file. 

and then run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
